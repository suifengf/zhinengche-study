# 雁过留痕学习

## 项目结构

### 📚 Libraries - 库文件夹

包含所有通用驱动程序、设备驱动和基础库文件，是项目的核心依赖库。

**主要内容：**

- **zf_common/** - 通用库
  - 时钟管理、调试、FIFO 队列、字体、中断、类型定义等基础功能
  
- **zf_driver/** - 底层驱动
  - ADC、GPIO、PWM、UART、SPI、USB CDC、定时器等硬件驱动接口
  
- **zf_device/** - 设备驱动
  - 各类传感器和模块：IMU（660RA/RB/RC）、IPS 液晶屏、摄像头、Wi-Fi 模块、蓝牙等
  
- **zf_components/** - 组件
  - 上位机助手、USB 库等高级组件

**版本信息：** 当前库版本 V3.2.0（2026-03-12）

---

### 🔧 temp - 样板模板

项目模板/样板文件，为新项目提供基础代码框架。

**包含：**
- `main.c` - 最小化的主程序框架
- `isr.c/isr.h` - 中断处理程序模板
- 基础的 MDK-ARM 项目配置

**用途：** 快速创建新的单片机应用项目

---

### ⚙️ Motor1 - 电机控制项目

基于库文件进行的具体应用实现，演示如何使用库中的驱动和功能。

**功能：**
- PWM 伺服电机控制（90° 角度）
- 多通道 PWM 输出控制（6 路电机/执行器）
- Wi-Fi SPI 模块通信集成
- TCP 连接和数据传输

**关键配置：**
- 系统时钟：96MHz
- 调试串口：已启用
- Wi-Fi 模块：自动连接到指定 TCP 服务器

---

## 快速开始

1. **使用样板创建新项目**
   - 复制 `temp/` 文件夹作为新项目的起点
   - 修改 `userheadfile.h` 配置硬件引脚

2. **参考 Motor1 实现**
   - Motor1 是完整的应用示例
   - 展示如何集成库文件进行复杂控制

3. **编译环境**
   - 使用 MDK-ARM（Keil uVision）
   - 确保链接了 `libraries/` 中的所有源文件
